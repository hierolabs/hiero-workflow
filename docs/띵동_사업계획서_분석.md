## 1. Executive Summary

- **팀명**: 띵동(THINGDONE)
- **아이템명**: 단기임대 AI 운영 어시스턴트
- **의미**: "무엇이든 띵(THING)하면 동(DONE)한다"
- **산업분야**: 공유숙박 / PropTech / 업무자동화 SaaS

### One-liner
> "카카오톡·엑셀로 청소를 수동 배정하는 공유숙박 다물건 운영자의 하루 1~2시간 반복 업무와 정산 오류 문제를 해결하기 위해 예약 연동 기반 청소 자동 배정·정산 자동화를 제공하는 앱"

### 산출물 (Odyssey-X 8주 목표)
- 모바일 앱 1종 MVP
- 청소 배정 자동화
- 유지보수 이슈 관리
- 추가비 기록 및 정산 자동화

## 2. Problem (문제 인식)

### 타겟 고객
- 에어비앤비, 삼삼엠투, 리브애니웨어, 아고다 등에서 **10개 이상 숙소 운영하는 다물건 운영자 및 위탁운영 업체**
- 매일: 청소 배정, 연장 예약 확인, 유지보수 대응, 추가비 정산, 인력 커뮤니케이션

### 핵심 문제
1. 운영 업무가 **하나의 시스템으로 연결되어 있지 않음**
2. Hostex → 캘린더 → 카카오톡 → 엑셀로 **데이터 분산**
3. 숙소 수 증가 → 하루 **1~2시간 반복 업무** + 청소 누락 + 중복 배정 + 정산 오류
4. 매출 플랫폼은 있지만, **현장 운영(청소·수리·정산) 시스템은 아날로그**

### 기존 대안의 한계
- Hostex + 캘린더 + 카카오톡 + 엑셀 조합
- 운영자 개인 숙련도에 과도하게 의존
- 확장할수록 소통 비용 증가
- "당장 돌아가게 하는 방식"일 뿐, 확장 가능 시스템 아님

## 3. Solution (실현 가능성)

### 제품 개요
띵동은 **예약 관리 앱이 아니라 운영 실행 앱**:
1. 다중 플랫폼 예약 연동 → 체크아웃 숙소 자동 추출
2. 연장 예약 + 레이트 체크아웃 반영 → 청소 대상 선별
3. 권역 코드 기반 청소 인력 자동 배정
4. 비포/애프터 사진 관리
5. 유지보수 이슈 등록
6. 추가비 기록 (소모품, 세탁, 배달)
7. 주간 정산 자동 계산

### 7대 핵심 기능
1. 예약 자동 연동
2. 청소 자동 배정
3. 배정 메시지 생성
4. 비포/애프터 사진 관리
5. 유지보수 트래킹
6. 추가비 원클릭 기록
7. 주간 정산 자동화

### 차별성
- **실제 100개 숙소 운영 경험**에서 출발한 서비스
- 현장에서 반복 검증된 프로세스를 제품으로 전환
- 서울시립대 도시공학 박사 5명 + 한양대 석사 1명
- 단순 청소 앱이 아니라 **청소+유지보수+추가비+정산 통합**
- 공공일자리 연계 가능성 (고령자·은퇴자 청소 인력)

### 개발 계획 (8주)
- 1~2주: 요구사항 정리, 화면 설계, 데이터 구조
- 3~4주: 예약 연동, 청소 판단, 배정 로직
- 5~6주: 유지보수, 추가비, 정산 자동화
- 7~8주: 현장 테스트, 피드백 반영, 데모데이

## 4. Scale-up (성장 전략)

### 비즈니스 모델 (월 구독형 SaaS)
| 구간 | 월 요금 |
|------|---------|
| 10~29개 숙소 | 29,000원 |
| 30~99개 숙소 | 59,000원 |
| 100개 이상 | 99,000원 |

향후: 위탁운영사 B2B, 지자체 B2G 확장

### 초기 고객 획득 (3단계)
1. **자체 검증**: 대표자 100개 숙소에 우선 적용 → 수치화 (배정 시간 단축, 누락 감소)
2. **네트워크 확산**: 다물건 운영자 + 위탁운영사 → 무료 베타 → 유료 전환
3. **커뮤니티 확산**: 사례 기반 영업 ("하루 몇 시간 절감" 실익 제시)

### 로드맵 (1년)
- 종료 직후: 현장 적용 + 안정성 검증
- 3개월: 베타 고객 확보 + 고도화
- 6개월: 유료 전환 + 상용 서비스
- 1년: 다중 플랫폼 확대 + 다자간 운영 구조 + 프리시드/시드 투자유치

## 6. Vision (비전)

### 지원 동기
- 단순 아이디어가 아니라 **실제 운영 현장에서 반복 확인된 문제 해결**
- 8주 실전형 프로그램에서 빠르게 제품화 + 검증

### 궁극적 비전
> 단기임대 운영자의 반복 업무를 줄여주는 앱 → **주거 운영 전반의 실행 인프라**

- 초기: 청소·유지보수·정산 자동화
- 중기: 청소 인력 체계적 운영, 고령자 일자리 창출
- 장기: 지자체 공공-민간 협력 거버넌스
- 글로벌: 표준형 운영 솔루션

### 데모데이까지 필요한 것 (2026.06.27)
1. **사진 업로드** (비포/애프터)
2. **추가비 입력** (소모품, 택시비, 세탁비)
3. **카카오톡/SMS 배정 메시지 연동**
4. **KPI 대시보드** (자동 생성률, 완료율, 정산 정확도)
