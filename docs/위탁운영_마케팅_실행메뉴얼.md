## 👥 타겟 고객 4가지 (우선순위 순)

### 1️⃣ 개별 호스트 (에어비앤비 운영자)
- 찾는 곳: 에어비앤비 후기 댓글, 당근마켓, 오늘의집
- 특징: 이미 숙소 있음, 관리에 지침
- 접근 방식: "수익 UP" 강조
- **우리 목표의 60% (120채)**

### 2️⃣ 건물주 (수십 채 보유)
- 찾는 곳: 부동산 중개소, 네이버 부동산, SNS
- 특징: 공실이 많음, 관리 경험 없음
- 접근 방식: "공실을 수익화하세요" 강조
- **우리 목표의 25% (50채)**

### 3️⃣ 중개인 (부동산 에이전트)
- 찾는 곳: 부동산 카톡 그룹, 전문 커뮤니티
- 특징: 고객 많음, 소개 중개수수료 원함
- 접근 방식: "고객소개 시 수수료 드림" 강조
- **우리 목표의 10% (20채)**

### 4️⃣ 소규모 투자자 (5~10채 구매 고려)
- 찾는 곳: 투자자 커뮤니티, 창업 카페
- 특징: "운영 걱정 없이 투자만" 원함
- 접근 방식: "운영은 우리, 수익은 당신" 강조
- **우리 목표의 5% (10채)**
