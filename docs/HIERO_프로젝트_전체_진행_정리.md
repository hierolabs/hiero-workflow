## 🚀 현재 진행 상황 (Step-by-Step)

### 현재 위치: 1주차 시작 (로컬 개발 환경 구축)

**이미 한 것:**
```
✅ npm 초기화 (package.json)
✅ PostgreSQL 테이블 설계
✅ API 엔드포인트 설계
✅ 데이터베이스 스키마 작성
```

**현재 문제 및 해결:**
```
❌ 문제: "npm run dev" 스크립트 없음
✅ 해결: package.json에 "dev" 스크립트 추가
        → nodemon 설치

❌ 문제: PostgreSQL 명령어를 터미널에 직접 실행
✅ 해결: psql -U postgres로 접속 후 SQL 실행

❌ 문제: .env 파일 없음
✅ 해결: .env 파일 생성 및 DB 정보 입력

결과: src/index.js 구현 완료 (기본 API 3개)
```

**다음 단계:**
```
1. npm run dev 실행 (서버 시작)
2. curl/Postman으로 API 테스트
3. 자동화 로직 추가 (상태 변경 시 다른 팀에 자동 알림)
4. 더 많은 API 추가 (댓글, 첨부파일 등)
5. 프론트엔드 개발로 이동 (2주차)
```
