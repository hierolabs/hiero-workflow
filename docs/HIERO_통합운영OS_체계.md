## 💰 재무 구조

### 수익 흐름

```

  ↓ (매달 입금)

  ↓

  ↓
호스트 몫 (85%) + HIERO 몫 (15%)

예시 (월 매출 300만원):
300만원 
 - 60만원 (청소비 20%)
 - 45만원 (HIERO 수수료 15%)
 = 195만원 (호스트 수령)

우리 수익: 45만원 × 300채 = 월 1.35억원
비용: 팀비 5,000만원 + 청소비 6,000만원
순이익: 월 1.85억원
```

### 월별 캐시플로우

```
1월: 신규 30채 (초기 설정비 발생)
  수입: 기존 100채 × 200만원 = 2억원
  지출: 팀비 + 청소비 + 신규 세팅비
  순이익: 1.5억원

2월~3월: 점진적 증가
  수입: 100채(기존) + 55채(신규) = 155채 × 150만원(평균)
  지출: 팀비 + 청소비 (점차 안정화)
  순이익: 1.8억원

4월~5월: 안정화
  수입: 100채 + 155채 = 255채 × 160만원
  지출: 안정화
  순이익: 2.3억원

6월 이후: 풀 가동
  수입: 300채 × 180만원 = 5.4억원
  지출: 팀비(안정) + 청소비
  순이익: 3.5억원 (연 42억원)
```

## 🤖 Multi-AI OS (자동화 엔진)

### 역할 분담

```
Claude (코드 작업자)
→ 숙소 상태 진단 스크립트 작성
→ 보고서 데이터 처리
→ 상품 설명문 생성
→ 고객 응대 템플릿 작성

GPT (설계자, 검토자)
→ 가격 전략 컨설팅
→ 마케팅 메시지 검수
→ 리스크 판단
→ 최종 결과물 검증

Gemini (대용량 분석가)
→ 긴 텍스트 분석 (1시간 이상의 로그)
→ 이전 데이터 통합 분석
→ 트렌드 예측

Router (자동 선택)
→ 어떤 AI를 언제 쓸지 자동 결정
→ 비용 최적화 (저비용 AI 우선)
→ 결과 검증 (자동 검수)
```

### 자동화 프로세스

```
08:00 - 일일 자동 리포트 생성
  HIERO DB → Claude → 일일 요약 → CEO 메일

10:00 - 청소 스케줄 최적화
  예약 DB → Claude → 청소팀 일정 추천 → 팀장 확인

12:00 - 가격책 조정
  시장 데이터 + 경쟁사 가격 → GPT → 추천 가격 → 수동 승인

14:00 - 고객 응대 템플릿 생성
  예약 + 숙소 정보 → Claude → 개인화 메시지 → 자동 발송

16:00 - 주간 보고서 (매주 금요일)
  일주일 데이터 → Gemini(분석) → GPT(정리) → 리포트 생성

20:00 - 이상 감지
  모든 숙소 데이터 → Claude → 비정상 물건 감지 → 알림 발송

23:00 - 월간 경영진 보고서 (매월 1일)
  월간 데이터 → Gemini(분석) + GPT(검증) → 대시보드 업데이트
```
