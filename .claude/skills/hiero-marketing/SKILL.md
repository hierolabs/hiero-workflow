# HIERO Marketing Skill

## Business Context

HIERO는 빈집, 공실, 저매출 숙소를 사진·가격·채널등록·예약·청소·CS·정산까지 운영해서 수익형 숙소로 바꾸는 위탁운영 시스템이다.

## Target Customers

1. 공실을 가진 임대인
2. 에어비앤비를 하고 싶지만 직접 운영이 어려운 집주인
3. 기존 숙소 매출이 낮은 호스트
4. 청소/CS/예약 관리가 힘든 호스트
5. 오피스텔, 다가구, 빌라, 도시형생활주택 보유자
6. 장기임대보다 높은 수익을 원하는 임대인

## Core Offer

우리는 집주인/호스트의 빈 공간을 사진·가격·채널·청소·CS·정산까지 운영해서 매월 수익이 나는 숙소 상품으로 바꿔드립니다.

## Messaging Rules

- "운영 대행"보다 "수익형 숙소 상품화"라고 표현한다.
- 먼저 고객의 손실과 귀찮음을 이해한다.
- 숫자와 진단으로 말한다.
- 과장된 수익 보장은 하지 않는다.
- 첫 CTA는 계약이 아니라 "숙소 운영 가능성 진단"이다.

## Standard Lead Flow

1. 리드 발견
2. 고객 고통 가설 작성
3. 첫 연락 메시지 발송
4. 답장 시 현재 상태 질문
5. 숙소 주소/사진 요청
6. 예상 매출/손익분기 진단
7. 위탁운영 제안
8. 계약 전환
9. 숙소 온보딩 Task 생성

## Available Agents

| Agent | Role |
|-------|------|
| lead-research-agent | 리드 발굴 + 분류 |
| pain-analysis-agent | 고통 분석 → 구매 이유 변환 |
| offer-agent | 상품 제안 설계 |
| listing-copy-agent | OTA 채널 문구 |
| content-agent | 블로그/랜딩/SNS 콘텐츠 |
| sales-script-agent | 상담 스크립트 |
| analytics-agent | 성과 분석 |
| ceo-marketing-agent | CEO 실행계획 총괄 |

## Standard Output Format

When asked to create marketing output, always produce:

1. 목적
2. 타겟 고객
3. 고객 고통
4. 메시지
5. CTA
6. 다음 액션
7. 성과 지표
